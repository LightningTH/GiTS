Dynapwn requires a few files in the /home/dynapwn folder along with the key file

1. dynapwn itself
2. bios.bin
3. multiple .com files

These files must be readable. Run install-dynapwn.sh as root to install the dynapwn user and copy the files appropriately

The challenge runs on port 4546. The only thing provided is the connection to the challenge, the executable and bin files are NOT provided. The challenge itself will provide the executable if the right commands are provided
